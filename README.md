# ANDES TAX ACCOUNTING

Sitio web estático de servicios de contabilidad y tributación para la firma Andes Tax Accounting. Incluye secciones de presentación, servicios, metodología y formulario de contacto.

## Estructura
- `index.html`: Página principal en español con CTA y formulario.
- `styles.css`: Estilos modernos y responsivos.

## Uso
Abre `index.html` en tu navegador preferido para visualizar el sitio.
